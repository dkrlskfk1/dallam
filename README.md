# API 문서

## API 목록

| HTTP 메서드 | URL                | 설명                  | 요청 본문        | 응답 본문        |
|-------------|--------------------|-----------------------|------------------|------------------|
| GET         | `/api/hello`       | Hello 메시지를 반환   | 없음             | `{ "message": "Hello" }` |
| POST        | `/api/create`      | 새로운 데이터를 생성 | `{ "name": "A" }` | `{ "id": 1, "name": "A" }` |
| DELETE      | `/api/delete/{id}` | 데이터를 삭제         | 없음             | `{ "message": "삭제 완료" }` |
